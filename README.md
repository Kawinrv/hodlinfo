# Hodlinfo 

This project fetches the top 10 cryptocurrency ticker data from the WazirX API, stores it in a MySQL database, and displays it on a web page that updates every 10 seconds.
## Build with

- HTML
- CSS
- Node.js
- MySQL

## Prerequisites

- Node.js
- MySQL
